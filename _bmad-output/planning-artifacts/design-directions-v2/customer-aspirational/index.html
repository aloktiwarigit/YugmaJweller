Volume II · Customer Aspirational

सौंदर्य की छह दिशाएँ Six aesthetic directions for a platform that refuses to look like a platform.

Goldsmith is a white-label platform. On every customer surface, only the jeweller's brand appears — the platform itself is invisible. These six directions are each a fully-committed answer to the question: what should a traditional Hindi-belt jeweller look like online to a 28-year-old software engineer in Lucknow shopping for her sister's wedding?

Each direction ships as three production HTML files — home, pdp, collection — totalling 20,810 lines of HTML + CSS + minimal JS. Photography, typography, motion, and motion-reduced fallbacks are all real. Tenant primary colour, accent, logo, name, domain, city are all CSS variables at :root, swappable per jeweller without touching markup.

II · White-label proof

एक दिशा, दो ज्वैलर One direction, two tenants — rendered live.

Goldsmith's non-negotiable: on every customer surface, only the jeweller's brand appears. Tenant primary, accent, name, city, and domain are CSS variables at :root. Swapping a tenant should preserve the aesthetic, not break it. Below, Direction Nº 04 · Gold Leaf Luxe is rendered live in two tenants — the anchor jeweller आँचल ज्वैलर्स of Ayodhya, and a mock second tenant राज ज्वैलर्स of Kanpur. Same markup. Same aesthetic grammar. Different colour seed and logo-mark.

Tenant · Aanchal आँचल ज्वैलर्स · अयोध्या
Tenant · Raj राज ज्वैलर्स · कानपुर

Because --tenant-primary is the load-bearing variable (gold-leaf accent stays fixed, doing most of the trust signalling), moving the seed from forest #14261C to sapphire #0D1B2A preserves hierarchy, contrast ratios, and the couture voice. The same exercise works on Directions 1, 3, 4, 6 identically. Directions 2, 5 swap differently — accent-led — which is documented in each file's :root header.

Note: if you're viewing this from a file:// URL in a browser that blocks cross-document iframe access, the sapphire override may not render here. Open direction-4-gold-leaf-luxe/home.html directly in devtools and change --tenant-primary on :root to see the swap live.

Creative peer opinion

मेरी सिफारिश If you ask me which to take forward.

Primary: Direction Nº 05 · Hindi-First Editorial. It's the only direction here that could not have been produced by a Western agency brief-reshuffled for India. Yatra One at 14–18rem composing with imagery — overlapping, breaking grid, scale-on-scroll — is a genuine visual moat. CaratLane, Tanishq, and Kalyan all treat Hindi as translation; this treats Devanagari as art. That's a competitive asset, not a cosmetic choice.

Close second: Direction Nº 03 · Heritage Paper. Paper-turbulence texture, letterpress Fraunces + Tiro pairing, wax-seal HUID ceremony. Earns craft-cult credibility for heritage jewellers specifically. At 5,426 lines, it's also the most thoroughly executed of the six — every edge case considered.

Commercial safety: Direction Nº 04 · Gold Leaf Luxe. If the anchor jeweller is nervous about "too arty," this is the Cartier-couture floor that swaps cleanly to other jewel tones without the direction breaking. Proven forest → sapphire swap.

  • 01   Pick primary + backup with the anchor jeweller in person — show this index on an iPad, let them scroll.
  • 02   Lock one direction; port its design tokens into the UX design-system file as the v2 `--tenant-*` baseline.
  • 03   Build the React Native / Next.js component library against that direction only — avoid trying to keep multiple alive post-choice.
  • 04   Re-commission the two weakest directions as rejected alternatives in the design archive (don't delete — reference material for future tenants).

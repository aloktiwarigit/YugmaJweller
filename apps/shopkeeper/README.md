# Goldsmith Shopkeeper — Expo

Dev Client app for the anchor jeweller shopkeeper. See `docs/runbook.md` for first-run setup.

## Quick start
1. Place required fonts per `assets/fonts/FONTS-TODO.md`.
2. Download `google-services.json` from Firebase console (`goldsmith-dev`) and place at project root.
3. `pnpm install` at repo root.
4. `pnpm --filter @goldsmith/shopkeeper start`.

## Screens
- `(auth)/phone` — Phone entry (Task 19)
- `(auth)/otp` — OTP verification (Task 19)
